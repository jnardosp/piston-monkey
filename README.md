# PistonMonkey
PistonMonkey is an AI-driven roadside assistance project focused on bringing expert mechanical knowledge to the edge. By fine-tuning small, highly efficient language models on technical workshop manuals, we aim to provide a "mechanic in your pocket" that works entirely offline on mobile devices.

This repository documents the journey from raw manual scraping to deploying a functional, fine-tuned model via Unsloth and Hugging Face, optimized for the constraints of mobile hardware.
